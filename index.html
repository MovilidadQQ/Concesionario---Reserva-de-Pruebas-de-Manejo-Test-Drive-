Backend simuladoAlgoritmo de tiempo de espera

calcularTiempoEspera() suma el tiempo promedio del tipo de servicio de cada vehículo que está delante en la cola. No es un valor fijo: si el auto de adelante tiene "Encerado" (más lento), el que ingresa después ve una espera mayor que si tuviera "Lavado Exterior". Al atender o retirar un vehículo, se recalculan todas las posiciones siguientes.

function tiempoPromedio(tipoServicio) {
  return DB.servicios.find(s => s.tipo === tipoServicio).minutos;
}

function calcularTiempoEspera(posicion) {
  // suma el tiempo de todos los vehículos delante de esta posición
  return DB.cola
    .slice(0, posicion)
    .reduce((total, v) => total + tiempoPromedio(v.servicio), 0);
}

function agregarVehiculo(patente, servicio) {
  DB.cola.push({ id: Date.now(), patente, servicio, ingreso: Date.now() });
  guardarDB();
}

function atenderSiguiente() {
  DB.cola.shift();      // el vehículo atendido sale de la cola
  guardarDB();          // el resto recalcula posición automáticamente al renderizar
}

Base de datos simuladaTablas servicios y cola

servicios guarda el tiempo promedio por tipo (catálogo fijo). cola es un array ordenado que funciona como una tabla con orden de llegada implícito — la posición de un registro en el array define su lugar en la fila, igual que un ORDER BY fecha_ingreso haría en SQL.
